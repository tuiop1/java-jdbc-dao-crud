**Java JDBC DAO CRUD Application**

This project is a console-based backend application written in Java.
It demonstrates database interaction using JDBC and the DAO pattern within a layered architecture.

The goal of the project is to practice structured backend development, separation of concerns, and clean interaction with a relational database.

**Features**

Create, read, update, and delete operations (CRUD)

JDBC-based database communication

DAO pattern implementation

Structured separation of application layers

Basic exception handling

**Technologies Used**

Java

JDBC

PostgreSQL

Git
